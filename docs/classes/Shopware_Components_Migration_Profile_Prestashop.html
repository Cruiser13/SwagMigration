Shopware_Components_Migration_Profile_Prestashop

Extends \Shopware_Components_Migration_Profile

Shopware SwagMigration Components - Prestashop

Base profile which all other profiles extend from. Basically offers a generic interface to the source profiles and checks if there is a method which returns a SQL query for the requested data. If so, in most cases the query will be executed and the query object will be returned. In some cases the result will directly be returned.
